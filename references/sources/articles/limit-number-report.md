# 莫氏鸡煲开始限号，有人驱车2000公里赶来，排队300多号|广东省|佛山市|老板娘|顺德|吐槽_手机新浪网

4月4日，早上6点就有客人登门。老板娘在社交账号吐槽，这么早过来排队，恐怖！莫氏鸡煲老板娘：“没办法，生无可恋啊。”

此前有网友发照片说老板“累瘫了，直接躺到了地上”，还说老板开小号自黑店里的鸡，又贵又难吃，想用自黑的歪招劝退客人。 不过老板娘发声辟谣，累瘫的照片是AI的，吐槽“鸡难吃”的账号不是老板开的小号，老板不会玩抖音。

有人驱车2000公里来吃，排队300多号

虽然表情包是假的，但老板辛苦是真的，不仅100个号秒空，自己每天还要剁50个鸡。就连被摇回家帮忙的“少东家”，每天都至少要砍70只鸡。老板女儿面对镜头都有些“崩不住”，“感觉这辈子都不想吃鸡煲了”。

记者探访发现，莫氏鸡煲中午就已经排到三百多号。面对火爆人气，老板喊话：让外地游客先吃，本地人错峰就餐，晚点再吃。老板儿子也坦诚喊话网友：店内空间实在有限，暂无招工计划，还贴心建议大家不妨去周边逛逛，品尝其他特色美食。

但是，老板越劝退，网友越“一身反骨”。有个早上10点就来排队，排了一两个小时的广西食客表示，“反正不差我一个，味道奶香奶香的，鸡也很脆，确实不错哦。”还吸引了山东友人专门开车2200多公里，来尝尝老板泡的药酒。

顺德陈村：莫氏鸡煲开始限号

今日（4月4日），据佛山顺德陈村镇人民政府公号“绿色陈村”消息，莫氏鸡煲要限号了。
