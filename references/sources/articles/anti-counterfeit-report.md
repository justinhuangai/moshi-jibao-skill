# 莫氏鸡煲打假：网上“鸡煲公主”账号并非家人，欢迎举报 | 极目新闻

极目新闻记者 丁伟

广东佛山顺德的莫氏鸡煲持续火爆，不少网友将老板莫叔的女儿称为“鸡煲公主”。近日网友发现，有多个名为“鸡煲公主”的社交账号经常发布莫氏鸡煲现场照，并在网上售卖鸡煲汤料包。4月15日，莫氏鸡煲回应，相关账号为冒充。

3月底，广东佛山顺德陈村镇绀现村莫氏鸡煲因美食博主的一条视频而意外走红，客流源源不断。为应对蜂拥而来的食客，莫氏鸡煲一家全员出动，老板莫叔的儿子、女儿都来帮忙。有网友发现老板的女儿和香港女星佘诗曼有几分相像，劝她原地出道，她也被叫作“鸡煲公主”。

网络截图

近日，有多个名为“鸡煲公主”的社交账号分享莫氏鸡煲的现场照片和视频，有的还在公开售卖鸡煲汤料包。4月15日，极目新闻记者给上述账号留言询问是否为莫氏鸡煲老板女儿本人，对方均未回应。

此前，莫叔父子曾多次公开表示，抖音“莫氏鸡煲”为唯一官方账号，目前他们未在网上卖任何东西，“大家别被骗”。

社交账号“鸡煲公主”是否为莫氏鸡煲账号？4月15日下午，莫氏鸡煲官方账号回应：“已经处理了一大批（冒充）账号，鸡煲公主不是我们的号，大家看到一个帮我们举报一个。”
